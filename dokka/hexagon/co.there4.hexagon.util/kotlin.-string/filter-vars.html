filterVars

fun String.filterVars(parameters: Map<*, *>): String

Filters the target string substituting each key by its value. The keys format is: #{key} and all occurrences are replaced by the supplied value.

If a variable does not have a parameter, it is left as it is.

Parameters

parameters - The map with the list of key/value tuples.

Return
The filtered text or the same string if no values are passed or found in the text.
